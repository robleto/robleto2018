Robleto

This is the 2018 design of the personal website for Greg Robleto. This site is a one-page site to share information about skills, interests and means of contacting Greg.

Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

Installing

This site is HTML, CSS and Javascript and has no requirements for installation of a virtual environment. This site is build using Codekit3 for linting, conversion and compression.

Deployment

This site is hosted by Dreamhost. Basic setup is all that was required.

Built Using

  • Photoshop - For creating the background and portfolio images
  • Atom - First time trying this IDE for coding and Markdown writing of this Readme
  • Transmit - Used for FTPing up to the Dreamhost server
